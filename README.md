# To Do List

A very simple To Do list app using Core Data.

### Preview
<img src="todoList.gif" width="200" />

### Features

- Persisting task list using Core Data
- Add, edit, delete and complete tasks

### Technologies

- MVC pattern
- Swift 5.0
- UIKit 
- Programmatic UI
- Core Data

### Requirements

- Xcode 10.3
- macOS 10.13.2
- iOS 12
